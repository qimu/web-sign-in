
# web-sign-in
a online sign-in form that replace traditional paper sign-in form at a event  
made for Temple of Enlightenment, NY
buildt on react-way-getting-started


## How to start

Run:  
```
npm install
npm start
```

## How to start code

Run:  
`npm run webpack-watch`
